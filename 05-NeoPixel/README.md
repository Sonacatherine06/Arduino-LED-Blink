# NeoPixel LED using Arduino Uno

## Components
- Arduino Uno
- NeoPixel Ring
- Jumper Wires

## Connections
- VCC -> 5V
- GND -> GND
- DIN -> D6

## Description
This project demonstrates controlling a NeoPixel LED ring using Arduino Uno. Different RGB color effects were implemented using the Adafruit NeoPixel library.
